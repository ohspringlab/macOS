Getting Started - Cooking with HungryMe

Adjust the font size in this help system only with: ( ⌘ + '+' ) and ( ⌘ + '-' )

HungryMe has a main window with Category elements on the left and Recipe elements on the right.

The display of a recipe's details is done as follows:

1. Select a recipe Category in the left table of the main window. Select "Browse All" if you wish to have all recipes listed.

2. Double click the desired recipe (or use Enter key), and a separate window will appear displaying the details for the selected recipe. Multiple recipes can be displayed simultaneously.

Recipe Text to Speech

Have the displayed recipe details read aloud by a speech synthesizer by clicking the Speech button at the lower right of the recipe details window.

The Best of Both Worlds

Please consider using your computer to gather digitized recipes into HungryMe and using iHungryMe and your iOS Device as your companion in the kitchen! Use iTunes File Sharing to ensure that your user-defined recipes remain in sync on all your devices.
